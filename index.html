Program Commands


‹ › 1 0 ?1 ?0 ?_ J _

Legend

‹ - Move Data Pointer Left
› - Move Data Pointer Right
1 - Set Data To 1
0 - Set Data To 0
?1 - If Data is 1, Switch Row
?0 - If Data is 0, Switch Row
?_ - If Data is Blank, Switch Row
J - Program Jumps To Begginning
_ - Do Nothing
